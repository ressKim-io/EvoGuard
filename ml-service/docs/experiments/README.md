## 실험 템플릿

새 실험 문서 작성 시 아래 구조를 따릅니다:

```markdown
# 실험: [제목]

> 날짜: YYYY-MM-DD
> 목표: [실험 목표]

## 1. 배경
- 기존 상태
- 문제점

## 2. 실험 설계
- 가설
- 방법

## 3. 실험 결과
- 정량적 결과 (표)
- 분석

## 4. 결론
- 성과
- 한계
- 향후 과제

## 5. 재현 방법
- 실행 명령어

## 6. 관련 파일
- 스크립트, 모델, 데이터 경로
```
